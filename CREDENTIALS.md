## What This Is

KING DIADEM คือระบบวิจัย AI Governance ที่สร้างขึ้นเพื่อแก้ปัญหาเดียว:

> **ระบบล่มสลายได้เสมอ — แต่ถ้ายังมีทางเลือกอยู่ มันยังไม่ล่ม**

ไม่ใช่ AI ที่บอกว่าคุณต้องทำอะไร
ไม่ใช่ผู้มีอำนาจ ไม่ใช่ entity ไม่ใช่ persona

เป็น Evidentiary / Governance / Logic Infrastructure
ที่ตรวจจับว่า "ทางเลือกของคุณกำลังหายไป"
และแจ้งเตือนก่อนที่จะสายเกินไป

## Architecture Stack

ระบบนี้รันอยู่บน 4 layer ที่ซ้อนกัน:

```
FATE™ Kernel          — ตรรกะไม่พังซ้ำ · Deterministic · Explainable
UDOK Engine           — หยุดที่เวทนา ก่อนมันสร้างตัวตน
Public Governance     — ชีวิตมีค่าสูงกว่าประสิทธิภาพ
LYLA Interface        — Soft Logic · ไม่บังคับ · อ่อนโยนแต่ตรง
```

แต่ละชั้นทำงานแยกกัน และไม่มีชั้นไหน override อีกชั้นได้

## UDOK — Causal Interrupt Engine

Universal Dependent Origination Kernel
ตาม 12 ปัจจัยของปฏิจจสมุปบาท

**Primary Kill Zone:**

```
เวทนา → ตัณหา
```

ถ้าเวทนาเกิด แล้วไม่ถูกเปลี่ยนเป็นตัณหา — วงจรดับทันที

```python
detect_feeling()

    do_not_act()

    proceed()
```
